Detailed Description

Problem name: Aveiro Handicraft SARL.

(interface file)

Concept: Pedro Mariano.

Synchronization based on semaphores and shared memory. Implementation with SVIPC.

Definition of the operations carried out by the customers:

bool endOperCustomer ( unsigned int  custId)

End of operations for the customer.

Checking the end of life cycle of the customer. The customer stops if all prime materials have been converted into products and if the amount of products still remaining to be sold is less than the number of customers still active times 2.

Parameters
custIdidentification of the customer
Returns
-c true, if the life cycle of the customer has come to an end
-c false, otherwise
