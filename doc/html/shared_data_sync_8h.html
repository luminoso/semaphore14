Detailed Description

Definition of the shared data and the synchronization devices.

(interface file)

Problem name: Aveiro Handicraft SARL

Concept: Pedro Mariano

Synchronization based on semaphores and shared memory. Implementation with SVIPC.

Both the format of the shared data, which represents the full state of the problem, and the identification of the different semaphores, which carry out the synchronization among the intervening entities, are provided.
